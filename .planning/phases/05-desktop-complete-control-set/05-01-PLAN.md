<objective>
Add the `IsSplitButton As Boolean = False` property to `XjRibbonItem` so that subsequent
plans (XjRibbonGroup factory method, XjRibbon mouse/draw logic) have a flag they can read
and write.

Purpose: A SplitButton is `ItemType = 2` (existing dropdown) WITH a body/arrow split. We
do not add a new ItemType — we add this flag. This keeps existing dropdown behavior
unchanged and is fully backward compatible (default = False).

block inserted between the existing `IsToggleActive` and `KeyTip` properties.
